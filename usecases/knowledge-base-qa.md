# 知识库问答助手：把散落信息变成可问可答

- 场景：围绕文档、笔记、知识库内容构建可对话助手
- 用户：团队、顾问、内容创作者、学习者
- 渠道：Feishu / Web / Discord / CLI
- 用到的能力：read / docs / memory / feishu_doc / message
- 难度：中级
- 效果：减少重复解释和翻资料时间

## 为什么值得做

很多知识并不缺，缺的是“在需要的时候快速取出来”。这类场景复用价值很高。

## 工作流

1. 选定知识来源
2. 规定回答范围与口径
3. 让 Agent 在回答时引用来源或明确不确定性
4. 持续补充高频问答

## 踩坑

- 不做范围约束，容易回答得很飘
- 如果没有引用机制，用户很难信任答案
