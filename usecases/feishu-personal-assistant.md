# Feishu 私人助理：把 OpenClaw 变成你的日常协作中枢

- 场景：通过 Feishu 与一个长期陪跑的 AI 助理协作，处理提醒、待办、信息查询与跟进
- 用户：知识工作者、创始人、顾问、内容创作者
- 渠道：Feishu
- 用到的能力：message / memory / feishu_doc / feishu_bitable / cron / tools
- 难度：入门
- 效果：把“想到一件事就记一下”升级成“有人持续帮你盯着、整理、推进”

## 为什么值得做

很多 AI 工具是一次性问答，但真正高频的价值来自持续协作。

把 OpenClaw 接到 Feishu 之后，AI 不再只是一个聊天窗口，而是可以逐步成为：

- 任务整理器
- 信息中枢
- 个人参谋
- 主动提醒系统

## 工作流

1. 在 Feishu 中接入 OpenClaw
2. 让 Agent 读取你的基础偏好、工作目标与长期任务
3. 日常通过自然语言丢任务、问问题、让它整理资料
4. 让 Agent 把重要事项同步到文档 / Bitable
5. 用 heartbeat 或 cron 做周期性提醒与回顾

## 关键 Prompt / 配置

```txt
你是我的长期协作助理。
你的默认工作方式是：
1. 先理解我要解决的问题
2. 如果能结构化，就帮我结构化
3. 如果能形成待办，就帮我拆成下一步
4. 如果是长期事项，提醒我建立跟踪机制
5. 给我结论时尽量直接，不要空话
```

## 踩坑

- 如果只是“接了聊天”，但没有记忆、文档、定时机制，价值会很快掉下去
- Bitable 适合结构化数据；普通 Sheet 反而更难自动化
- 长期体验很大程度取决于系统提示词和记忆管理
