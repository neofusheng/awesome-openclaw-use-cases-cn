# Founder Chief-of-Staff Agent：给创始人的 AI 参谋台

- 场景：服务创始人的任务推进、信息整理、战略讨论、提醒跟进
- 用户：创始人、顾问、高管
- 渠道：Feishu / Telegram / WhatsApp
- 用到的能力：memory / message / docs / cron / structured workflows
- 难度：高级
- 效果：把零散对话升级成持续决策协作系统

## 为什么值得做

创始人最缺的不是一个“回答问题的 AI”，而是一个能持续跟进上下文的参谋角色。

## 工作流

1. 沉淀长期目标、关键项目、重要关系
2. 把每天输入变成待决策事项/待推进事项
3. 做日/周节奏回顾
4. 对重要议题给出结构化判断与备选方案

## 踩坑

- 如果没有长期上下文，这类助手很难体现差异化
- 必须区分“建议”“事实”“判断”，否则很容易失真
