# Heartbeat 主动助理：不等你开口，AI 也会适时提醒你

- 场景：通过 heartbeat 机制，让 Agent 在合适的时候主动检查并提醒事项
- 用户：忙碌的创始人、顾问、知识工作者
- 渠道：Feishu / Telegram / Discord
- 用到的能力：heartbeat / memory / cron / message / docs
- 难度：中级
- 效果：从“被动问答”升级到“轻主动协作”

## 为什么值得做

真正有价值的助理，不是每次都等你想起来再问，而是在你快忘的时候提醒你、在该同步时同步你。

## 工作流

1. 设定 heartbeat 检查规则
2. 限定只关注重要事项，避免打扰
3. 根据日程、未完成任务、最近上下文做轻提醒
4. 重要信息再转入待办/文档

## 踩坑

- 主动不等于高频打扰
- 没有优先级机制时，用户很容易烦
