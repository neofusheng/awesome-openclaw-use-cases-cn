# WhatsApp 客户跟进助手：把私域沟通变成可持续流程

- 场景：围绕客户咨询、跟进、提醒、资料发送建立 AI 辅助流程
- 用户：顾问、销售、私域运营、服务型创始人
- 渠道：WhatsApp
- 用到的能力：message / memory / docs / cron / structured follow-up
- 难度：中级
- 效果：减少漏跟进，提升服务连续性

## 为什么值得做

很多成交不是输在产品，而是输在跟进断了。AI 很适合做跟进节奏、要点整理、下一步提示。

## 工作流

1. 记录客户状态与关键上下文
2. 根据沟通结果生成下一步建议
3. 设定跟进提醒
4. 持续沉淀常见问题与回复素材

## 踩坑

- 涉及外发消息时，要明确哪些可以自动、哪些必须人工确认
- 客户沟通要重视语气和边界，不能太像机器人
