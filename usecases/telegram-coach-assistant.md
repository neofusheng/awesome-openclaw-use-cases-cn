# Telegram 陪跑助理：把 AI 放进最高频的聊天入口

- 场景：在 Telegram 中持续和 AI 协作，做提醒、复盘、问答、任务推进
- 用户：自由职业者、创始人、学习者、出海从业者
- 渠道：Telegram
- 用到的能力：message / memory / cron / tools / sessions
- 难度：入门
- 效果：把“偶尔用 AI”变成“每天都在和 AI 一起推进事情”

## 为什么值得做

Telegram 是很多人最自然的高频入口。把 OpenClaw 接进去，AI 的价值会从一次性问答变成持续协作。

## 工作流

1. 接入 Telegram bot
2. 设定长期目标与协作方式
3. 日常丢问题、任务、灵感
4. 用 cron/heartbeat 做回顾与提醒
5. 让 Agent 形成持续陪跑关系

## 关键 Prompt / 配置

```txt
你是我的长期陪跑助理。遇到目标模糊时，帮我先拆解；遇到任务过大时，帮我先给下一步；遇到拖延时，提醒我回到最小行动。
```

## 踩坑

- 只聊天不形成跟进机制，体验会很快变弱
- 要求“像朋友聊天”和“像项目经理推进”这两种模式最好区分开
