# AI 研究日报：把信息噪音压缩成真正有价值的更新

- 场景：跟踪 AI 资讯、论文、产品动态，并整理成日报/周报
- 用户：AI 从业者、投资人、顾问、内容创作者
- 渠道：Feishu / Telegram / Web / CLI
- 用到的能力：web / docs / summarization / message / memory
- 难度：中级
- 效果：减少信息过载，提高输入质量

## 为什么值得做

很多人不是信息太少，而是信息太多。Agent 的价值是做过滤、归纳、判断优先级。

## 工作流

1. 设定关注主题与来源
2. 抓取更新并去重
3. 按“重要性 / 可行动性 / 新颖性”排序
4. 输出日报/周报

## 踩坑

- 纯摘要没有价值，必须加“为什么值得看”
- 不做去重，很快会变成信息垃圾桶
