# 日历提醒 Agent：把提醒从“到点响一下”升级成“有上下文的提醒”

- 场景：围绕会议、出行、截止日期生成更聪明的提醒
- 用户：高频开会的人、项目负责人、顾问
- 渠道：Feishu / Telegram / WhatsApp
- 用到的能力：cron / calendar context / memory / message
- 难度：入门
- 效果：提醒不只是通知，而是帮助你更好地准备下一步

## 为什么值得做

普通提醒只告诉你“时间到了”，好的提醒会告诉你“接下来该准备什么”。

## 工作流

1. 获取日历事件
2. 识别重要会议/截止事项
3. 提前适当时间提醒
4. 附带相关背景、文档、待准备事项

## 踩坑

- 提醒太多会麻木
- 必须把重要性和时间窗口设好
